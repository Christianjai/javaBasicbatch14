Compiler always checks our code before giving it to JAVA to execute it.
Compiler can initialize variables to its default values:
- int =0;
- double=0.0;
- boolean=false;
- String and other non primitive=null;


SWITCH CASE - Statement executes one statement from multiple values/condition

SYNTAX

switch (variable) {

	case value1:
		code A;
		break;
	case value2:
		code B;
		break;
	default:
		code C;
		break;
}
